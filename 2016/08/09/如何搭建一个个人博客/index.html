如何搭建一个个人博客

发表于 2016-08-09   |   分类于 WEB综合   |  

1.前言

  • 现在写博客的人越来越少了，但是还是有一些人希望用博客记录一下自己的所思所想或学习笔记。其中一部分人选择了一些网站提供的博客服务，一些人或有自己的个性需求或想通过建站学习技术，希望搭建一个自己的博客。不管你出于什么目的，本文将帮助你搭建一个自己的博客。
  • 本文主要梳理方法，相关操作步骤由于官方文档和网上资料都很详尽，我就不在此啰嗦了。

    2.技术要点

  • 通常我们搭建一个网站需要以下几个步骤：
    • 注册一个域名。例如www.yourname.com.可以找代理商（例如万网等域名商）买，这样你就可以有一个你名字的域名了。
    • 买网站空间。存放你网站代码的地方。
    • 选择一个成熟的网站程序。如wordpress、ZBLog等等。
    • 解析域名，上传网站程序及资料。 将你的域名和空间绑定，这样人们就可以通过你的域名访问你的网站了。
  • 听上去很简单吧，这就是建立一个网站的步骤。而如此简单的原因主要归功于现在很多现成的成熟的开源程序。极大的简化了普通人写代码的难度，说白了就是大神造好了轮子和发动机，我们直接组装小汽车。但最后，随着技术提高，我们还是需要造更好的轮子的。
  • 基于这个建站步骤，我们只要选择用什么框架就好了，综合考虑来看，我们选择Hexo+Github pages的方式（有兴趣的可以搜索其他的方式）来建立博客。
  • Github Pages可以提供给我们一个免费稳定的空间，一个在github下的域名。并且我们可以用github来管理我们的博客，一举三得。hexo 提供给我们一个界面干净简洁的博客框架。这样，四个步骤都齐了，下面就按步骤组装吧。

    3.Hexo+Github Pages方式搭建博客

    （1）环境准备（根据自己电脑选择mac和win版本）

  • 安装node
    到Node.js官网下载相应平台的最新版本，一路安装即可。
  • 安装Git
    Git的客户端很多，可以用官方的，也可以用msysgit，根据自己电脑选择mac和win版本。
  • 安装编译软件
    sublime(推荐） Dreamweaver(初学者) webstorm,无论哪种，你会就行。

    （2）安装Hexo https://hexo.io/zh-cn/

  • 自己先cd到你想要安装的目录下（避免安装到自己不想安装的目录下）
  • npm install hexo-cli -g，安装Hexo
  • hexo init blog ，初始化（创建blog文件夹作为工作目录）
  • cd blog ，调整目录到blog文件夹下
  • npm install ，安装依赖包
  • hexo server ，生成静态页面
  • 浏览器输入http://localhost:4000 ，本地启动测试就可以看到效果。

    （3）注册Github, 建立仓库，部署

  • 注册Github https://github.com
  • 建立与你用户名对应的仓库，仓库名必须为 your_user_name.github.com
  • 添加SSH公钥

    （4）部署

  • 编辑_config.yml 配置文件(你安装目录下:\hexo下)。
    deploy:
    type: github
    repository: https://github.com/octivy（替换成你的github名）/octivy（替换成你的github名）.github.io.git
    branch: master
  • 编辑完成后保存，执行：
    hexo generate
    hexo deploy
